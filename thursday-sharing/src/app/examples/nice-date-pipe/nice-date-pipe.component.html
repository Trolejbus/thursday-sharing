Allows to display dates in nicer way, than simple format. Dates will be automatically updates over time. Also pipe is working with ChangeDetection.OnPush strategy. Pipe result can be also easly translated to other language (see translate method in NiceDateTimePipe class).

This is only example code, and should be tested before production.

See code at: github
